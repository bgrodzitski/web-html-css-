Глава 1 Создать сайт в стиле HTML5 используя заданную структуру согласно варианта. (Структура сайта дана в конце этого файла!!!) Для оформления содержимого сайта использовать CSS3. Заголовок сайта содержит фамилию, имя, отчество. Заголовок оформить в виде объемной надписи с использованием позиционирования и свойства z-index. Основное меню: «Форматирование, CSS, DOM, Кроссворд, String, Формы, Data, Object». Каждый элемент основного меню содержит несколько заданий, которые появляются при выборе основного пункта меню в виде «Подзадание навигация».
часть 2 Пункты «Подзадание навигация» для элемента основного меню «Форматирование»: • «Предварительно отформатированный текст»: используя тег предварительно отформатированного текста вывести стихотворение . • «Цитаты»: используя теги для цитат, задать 3 различных цитаты. • «Списки»: используя теги задания списков, задать 3 различных списка (нумерованный, маркированный, список определений). В качестве нумерованного списка использовать список литературы из любой курсовой работы. Каждый список должен иметь не менее 10 элементов. • «Таблицы»: представить информацию в виде таблицы (с объединением ячеек в строках или столбцах). Таблица должна содержать заголовочную, основную часть и footer. Для каждой структурной части таблицы задать различной цветовое и шрифтовое оформление. • «escape-последовательности»: вывести на страничку сайта код для создания таблицы с учетом форматирования. • «Закладки»: задать не менее двух переходов внутри страницы. • «Виджет»: создать простой открывающийся и закрывающийся виджет, то есть фрагмент краткого изложения, по щелчку на котором открывается панель с дополнительной информацией используя теги <details> и <summary>. • «Карта ссылок»: логотип оформить в виде карты ссылок, разбив изображение не менее чем на 3 многоугольника. При выборе заданной области должен осуществляться переход на некоторые пункты меню «Форматирование». В качестве рисунка можно взять рисунок из своего варианта. • «Плавающий фрейм»: создать плавающий фрейм и загрузить в него небольшой html-документ.
Глава 3 Подвал страницы сайта должен содержать сведения об авторских правах и адрес электронной почты.
